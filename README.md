# CLI multimatter

CLI App to interact mainly with Obsidian vaults.

⚠️ This is a work in progress.

## Install

To install dependencies:

```sh
bun install
```

## Run

Rename `.env.development` to `.env` and add your Obsidian vault path.

To run:

```sh
bun run dev
```

## Build

To build:

```sh
bun build
```
